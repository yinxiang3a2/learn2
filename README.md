# 项目概述
基于 Kotlin + XML 布局 的 Android 原生应用示例，主要实现 用户登录认证与个人中心管理，适合作为 Android 原生开发学习 Demo。
# 功能模块
## 1.登录页面（Login Screen）
  输入邮箱和密码进行身份验证。
  邮箱账号格式验证，密码可隐藏可观看。
  UI 特性：沉浸式状态栏、输入框焦点高亮、密码可见切换；
  社交登录按钮（微信/Apple），点击弹 Toast 提示。
  数据存储：SQLite 数据库存储账号密码，预埋测试账户。
## 2.注册页面（Register Screen）
  输入邮箱、密码及确认密码完成注册
  验证流程：完整性检查、邮箱格式、密码一致性、账号唯一性
  数据写入 SQLite 数据库
  注册成功自动返回登录页
## 3.个人中心页面（Profile Screen）
  显示用户信息卡片（邮箱、个性签名）
  功能列表（个人信息、收藏、历史、社区等）
